    java中有几种类型的流？

    1. Java中有几种类型的流？ JDK为每种类型的流提供了一些抽象类类以供继承，请说出它们分别是那些类？

    IO流用来处理设备之间的数据传输

    流的分类：

    1. 流按处理数据类型分为两种：字节流与字符流
    2. 流按数据流向分:输入流、输出流

    字符流的抽象基类：

    1. Reader用于读取字符流的抽象类
    2. Writer写入字符流的抽象类

    字节流的抽象基类：

    1. InputStream此抽象类是表示字节输入流的所有类的超类。
    2. OutputStream此抽象类是表示输出字节流的所有类的超类
